# next-to-go

Reat SPA application displaying races using this API https://api.neds.com.au/rest/v1/racing/?method=nextraces&count=10

## Getting Started

Install dependencies

```
npm install
```

Start dev server locally:

```
npm start
```

## Dependencies

- React
- axios
- material-ui/core
- react-countdown
